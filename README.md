# Personal Portfolio Website

## Giới thiệu

Đây là website portfolio cá nhân được xây dựng nhằm giới thiệu bản thân, kỹ năng và các dự án thiết kế.

Website được phát triển bằng HTML, CSS và có ứng dụng công cụ trí tuệ nhân tạo trong quá trình thiết kế giao diện và xây dựng nội dung.

---

# Công nghệ sử dụng

- HTML5
- CSS3
- GitHub Pages

---

# Công cụ AI đã sử dụng

## 1. ChatGPT
Sử dụng để:
- Hỗ trợ viết cấu trúc HTML/CSS
- Gợi ý bố cục portfolio
- Tạo hiệu ứng hover
- Responsive giao diện
- Thiết kế hình ảnh minh họa
## 2. Canva AI
Sử dụng để:
- Thiết kế layout portfolio
- Xây dựng ý tưởng giao diện



# Quá trình chỉnh sửa kết quả AI

Sau khi sử dụng AI để tạo ý tưởng và cấu trúc ban đầu, tôi đã thực hiện các bước chỉnh sửa và hoàn thiện:

- Điều chỉnh màu sắc phù hợp phong cách cá nhân
- Chỉnh sửa bố cục để dễ nhìn hơn
- Tối ưu responsive cho điện thoại
- Thay đổi typography và khoảng cách
- Tự chỉnh sửa nội dung giới thiệu và project

# Ý tưởng thiết kế

Website được thiết kế theo phong cách:
- Dark Modern
- Minimalist
- Creative Portfolio

Mục tiêu là tạo nên một portfolio hiện đại, dễ đọc, thể hiện được cá tính thiết kế và phù hợp với nhà tuyển dụng.
