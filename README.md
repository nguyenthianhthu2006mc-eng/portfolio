# Personal Portfolio Website

## Giới thiệu

Đây là website portfolio cá nhân được xây dựng nhằm giới thiệu bản thân, kỹ năng và các dự án thiết kế.

Website được phát triển bằng HTML, CSS và có ứng dụng công cụ trí tuệ nhân tạo trong quá trình thiết kế giao diện và xây dựng nội dung.

---

# Công nghệ sử dụng

- HTML5
- CSS3
- GitHub Pages

# Công cụ AI đã sử dụng

## 1. ChatGPT
Sử dụng để:
- Hỗ trợ viết cấu trúc HTML/CSS
- Gợi ý bố cục portfolio
- Tạo hiệu ứng hover
- Responsive giao diện
- Thiết kế hình ảnh minh họa
## 2. Canva AI
Sử dụng để:
- Thiết kế layout portfolio
- Xây dựng ý tưởng giao diện



# Quá trình sử dụng AI

Tạo cho tôi một website portfolio cá nhân bằng HTML và CSS theo phong cách modern dark aesthetic, cinematic và chuyên nghiệp dành cho Graphic Designer/UI UX Designer.

Yêu cầu:
- Giao diện responsive
- Tông màu tối hiện đại
- Font Poppins
- Hero section full màn hình
- Background hero có thể thay bằng ảnh cá nhân trong thư mục images/background.jpg
- Có lớp overlay tối để nổi chữ
- Có hiệu ứng background chuyển động nhẹ/parallax
- Từ phần About trở xuống dùng background khác trong images/background2.jpg phủ toàn bộ phần dưới
- Có hiệu ứng glassmorphism cho card
- Có hiệu ứng hover cho button/card
- Có hiệu ứng mưa rơi cinematic toàn trang
- Có smooth scroll
- Có animation mềm mại khi hover

Các section gồm:
1. Hero
2. About Me
3. Skills
4. Projects
5. Contact
6. Footer

PROJECTS:
